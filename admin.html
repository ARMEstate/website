One-time setup

This connects the admin to your website's storage. You only do this once per device — after saving, this page goes straight to the upload form.
How to get the key: GitHub → click your photo → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Name it "ARM admin", set expiry to 1 year, under "Repository access" pick Only select repositories → website, and under Permissions set Contents → Read and write. Generate, copy, paste here. (The key stays only in this device's browser.)
